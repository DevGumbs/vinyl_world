# Vinyl World - Prototype

This directory contains HTML/Tailwind CSS prototypes for the Vinyl World platform. These are static mockups used for design and layout purposes.

## ⚠️ Important Note

**This is a prototype only.** Most links and interactive elements do not function. Only the navigation menu links in the header are wired up to navigate between pages.

## Available Pages

- **`index.html`** - Lobby / Home page - (https://devgumbs.github.io/vinyl_world/prototype/index.html)
- **`dashboard.html`** - User Dashboard (https://devgumbs.github.io/vinyl_world/prototype/dashboard.html)
- **`collection.html`** - My Collection (https://devgumbs.github.io/vinyl_world/prototype/collection.html)
- **`shelf.html`** - Shelf View (https://devgumbs.github.io/vinyl_world/prototype/shelf.html)
- **`trade.html`** - Trade Center (https://devgumbs.github.io/vinyl_world/prototype/trade.html)

## Navigation

The header navigation menu on each page includes working links to:
- Home → `index.html`
- My Collection → `collection.html`
- Trade Center → `trade.html`
- Username → `dashboard.html`

Additionally:
- The "Shelf View" link on the dashboard page navigates to `shelf.html`
- The "Shelf" / "Gallery" toggle on collection and shelf pages switches between views
